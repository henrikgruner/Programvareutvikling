# Form elements

All of these fields must be inside a `formik` form.

## TextBoxField

```js
import { Field } from "formik";
import { TextBoxField } from "../../components/form";`

<Field name="firstname" component={TextBoxField} label="Fornavn" placeholder="Ola" />
```

## PasswordField

```js
import { Field } from "formik";
import { PasswordField } from "../../components/form";`

<Field name="text" component={PasswordField}/>
```

## EmailField

```js
import { Field } from "formik";
```
